O Desafio da Serpente

A serpente está com fome! Use a lógica de if / else if / else para guiá-la até a comida. O seu código será executado a cada nova tentativa!

    OBJETOS DISPONÍVEIS:
  • Serpente: `elementos.serpente`
  • Comida: `elementos.comida`
    AÇÕES DISPONÍVEIS:
  • Serpente: `acoes.moverParaDireita()`
  • Serpente: `acoes.moverParaEsquerda()`
  • Serpente: `acoes.moverParaCima()`
  • Serpente: `acoes.moverParaBaixo()`

Dica completa:
• Se `elementos.serpente.posicaoX < elementos.comida.posicaoX` → comida está à direita → `acoes.moverParaDireita()`
• Se `elementos.serpente.posicaoX > elementos.comida.posicaoX` → comida está à esquerda → `acoes.moverParaEsquerda()`
• Se `elementos.serpente.posicaoY < elementos.comida.posicaoY` → comida está acima → `acoes.moverParaCima()`
• Se `elementos.serpente.posicaoY > elementos.comida.posicaoY` → comida está abaixo → `acoes.moverParaBaixo()`

Você conseguiu!!

Como funciona?

O seu código no editor decide o próximo movimento da serpente. A cada passo, seu script é consultado para saber para onde ir. Use a lógica `if/else if/else` para criar uma estratégia e capturar a comida sem bater nas paredes!
Tenha em mente que X e Y são coodernadas do plano cartesiano.
X = horizontal
Y = vertical

Estrutura Sugerida

1. Compare posição X primeiro
2. Se a comparação de X der certo, compare agora Y
3. Use `else if` para cada condição
