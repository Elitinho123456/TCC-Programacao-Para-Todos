Fase 3/4:
O Desafio da Serpente

A serpente está com fome! Use a lógica de if / else if / else para guiá-la até a comida. O seu código será executado a cada passo que a serpente der!

  • Serpente: `elementos.serpente`
  • Comida: `elementos.comida`

Compare `serpente.posicaoX` com `comida.posicaoX` e `serpente.posicaoY` com `comida.posicaoY` para decidir qual ação tomar.

Se a posição X da serpente for menor que a da comida (`serpente.posicaoX < comida.posicaoX`), significa que a comida está à direita. Nesse caso, você deve usar `acoes.moverParaDireita()`.

Como funciona?

O seu código no editor decide o próximo movimento da serpente. A cada passo, seu script é consultado para saber para onde ir. Use a lógica `if/else` para criar uma estratégia e capturar a comida sem bater nas paredes!
