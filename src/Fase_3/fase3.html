Use a lógica de programação para ajudar o Pac-Man a navegar pelo labirinto. Evite fantasmas e colete as pílulas com comandos lógicos!

  • Sensores: `elementos.sensor.frente`, `direita`, `esquerda`
  • Ações: `acoes.virarDireita()`, `virarEsquerda()`, `continuarReto()`

Use condicionais para desviar de paredes e coletar pílulas. Por exemplo: if (elementos.sensor.frente === 'parede') { acoes.virarDireita(); } else { acoes.continuarReto(); }
