Desafio Final:
O Labirinto de Condições

O Pac-Man quer ajuda para coletar todas as pílulas! Use a lógica de if / else if para criar uma rota inteligente e desviar das paredes.

Ferramentas Disponíveis

Seu código tem acesso a dois objetos mágicos. Os detalhes de como usá-los estão nos comentários do editor!

  • sensores: Informa onde o Pac-Man está e para onde ele pode ir.
  • acoes: Contém as funções para mover o Pac-Man.

A lógica é: "SE a pílula está à direita E o caminho está livre, ENTÃO mova para a direita".

if (condicao_1 && condicao_2) { acao; }

Estratégia Chave

O segredo é combinar duas perguntas em uma só usando o operador && (E):

  • Onde está a pílula mais próxima?
  • O caminho naquela direção está livre?

Se a resposta para ambas as perguntas for "sim", você encontrou o movimento certo! Teste todas as quatro direções com if e else if.
