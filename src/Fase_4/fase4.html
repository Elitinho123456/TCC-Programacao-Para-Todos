A Rota Perfeita

O Pac-Man está faminto e quer ele colete todas as frutinhas do labirinto! Para isso, você precisará criar uma lógica inteligente usando as estruturas condicionais if (se) e else if (senão se). Seu código irá decidir para onde o Pac-Man deve se mover a cada passo, evitando as paredes.

Os Sensores do Pac-Man

Seu código recebe um objeto sensores com informações vitais em tempo real. As mais importantes para este desafio são:

  • sensores.pilulaMaisProxima: Informa as coordenadas (x, y) da pílula mais perto.
  • sensores.podeMoverPara...: Retorna true ou false se o caminho em uma direção (Cima, Baixo, Esquerda, Direita) está livre.

Pense na lógica em português: "SE a pílula está à direita E o caminho para a direita está livre, ENTÃO mova para a direita".

Em código, isso se parece com:
if (sensores.pilulaMaisProxima.x > sensores.pacman.x && sensores.podeMoverParaDireita) { ... }

Como Funciona?

O segredo para vencer é usar condições compostas. Uma condição simples seria apenas verificar se o Pac-Man pode ir para a direita. Uma condição composta é mais poderosa, pois verifica duas ou mais coisas de uma vez usando o operador && (E).

Estratégia Vencedora

A lógica mais eficiente é combinar essas duas informações. A cada passo, seu código deve perguntar, por exemplo: "A pílula mais próxima está acima de mim E eu posso me mover para cima?". Se a resposta for sim, execute a ação para subir. Caso contrário, teste as outras direções com else if.
