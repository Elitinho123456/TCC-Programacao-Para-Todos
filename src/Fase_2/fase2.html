Fase 2/4:
O Loop Sônico

O loop é acionado AUTOMATICAMENTE!

A velocidade do jogador aumenta com o tempo, mas não rápido o suficiente! Sua tarefa é usar o editor de código para dar um impulso inicial na velocidade do jogador.

Use o objeto `elementos` para alterar a velocidade inicial. O seu código será executado assim que você clicar em "Tentar Novamente".

  • Para definir a velocidade: `elementos.velocidadeAtual = 20;`
  • Velocidade Necessária: `elementos.velocidadeNecessaria` (somente leitura)

Desafio: Encontre o valor de velocidade inicial perfeito para passar pelo loop!

A velocidade aumenta com o tempo. Um impulso inicial entre `15` e `25` pode ser o suficiente. Experimente com valores diferentes! Lembre-se que a velocidade necessária é 40 km/h.
